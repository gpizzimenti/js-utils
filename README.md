# A collection of vanilla js helpers in ES module format:

 - **browserUtils.js** : 
	 - isTouchDevice
	 - isPortrait
	 - isSmallFormat
	 - browserData 
	 
 - **colorUtils.js:**
	 - HexToRGB
	 - HexToHSL 
	 - HSLToRGB
	 - invertColor
	 
  - **domUtils.js:**
	 - escapeHtml
	 - unescapeHtml 
	 
  - **execUtils.js:**
	 - nextFrame
	 - softExec
	 - debounce 
	 - throttle
	 
  - **geoUtils.js:**
	 - DecimaltoDMS
	 - DMSToDecimal
	 - formatDecimalToDMS 
	 - formatDMSToDecimal
	 - haversineDistance
