	 - isTouchDevice
	 - isPortrait
	 - isSmallFormat
	 - browserData

	 - HexToRGB
	 - HexToHSL 
	 - HSLToRGB
	 - HSLToHex
	 - invertColor
	 
  - **[domUtils.js](https://github.com/gpizzimenti/js-utils/blob/main/domUtils.js):**
	 - escapeHtml
	 - unescapeHtml 
	 
  - **[execUtils.js](execUtils.js):**
	 - nextFrame
	 - softExec
	 - debounce 
	 - throttle
	 
  - **[geoUtils.js](https://github.com/gpizzimenti/js-utils/blob/main/geoUtils.js):**
	 - DecimaltoDMS
	 - DMSToDecimal
	 - formatDecimalToDMS 
	 - formatDMSToDecimal
	 - haversineDistance
